# AWS-disaster-recovery-project
🔐Encrypted EC2 Backup Automation &amp; Disaster Recovery System (AWS)
## 📌 Overview
Automated disaster recovery solution using AWS EC2, EBS Snapshots, Lambda, and CloudFormation.

## 🏗 Architecture
- EC2 Instance
- Encrypted EBS volumes
- Snapshot backups
- Recovery mount server
- Lambda automation

## ⚙️ Technologies Used
- AWS EC2
- EBS & Snapshots
- Lambda (Python)
- CloudFormation
- Amazon Linux

## 🔄 Workflow
1. Production EC2 volume snapshot
2. Encrypted backup creation
3. Attach to recovery instance
4. Mount & verify data

## ✅ Outcome
Successfully automated disaster recovery with secure backup and fast restoration.
